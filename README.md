# Exotic-Folder-Tracker

The Exotic Folder Tracker is a programm, that tracks all .txt files in your folder.

If you first start the programm your askes to put your path in it, thats the path to your folder you want to Track .txt files in it!
Then the programm is gonna create a config.json file, thats where your path is saved so you dont have to put it in every time you use it.
